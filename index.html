Ingest Notes

Refresh Command

Run node scripts/harvest-nara-scout-search.js to refresh data/records.json and reports/nara-scout-volume28-candidates.json. Use node scripts/harvest-topic-records.js for the deeper focused pass through the named Scowcroft, NSC/DC, NSR, NSD, and transition collections. Use node scripts/harvest-whorm.js to refresh the WHORM case-file layer, and node scripts/harvest-chief-of-staff.js to refresh Chief of Staff files. Use node scripts/harvest-all-textual-collections.js to refresh the full Bush Library textual-collections sweep. Use node scripts/refine-memcon-telcon-only.js after any broad harvest to narrow the public chronology back to declassified memcons and telcons only.
