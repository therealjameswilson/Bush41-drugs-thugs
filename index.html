    Ingest Notes

    Refresh Command

    Run node scripts/harvest-presidential-conversations.js to refresh the public chronology from the Presidential Memcon and Telcon Files, then run node scripts/enrich-conversation-pdfs.js to measure pages and extract first-page verification metadata. Run node scripts/harvest-daily-diary-references.js to add same-day Presidential Daily Diary and Daily Backup control references for meetings and calls. The broader harvest scripts and reports remain in the repository for source discovery, while this site publishes declassified conversations plus explicit gap queues for compiler risk. Run node scripts/harvest-public-statements.js to refresh the GovInfo Public Papers reference section, then run node scripts/build-compiler-risk-data.js to rebuild event dossiers, gap lists, and the Public Papers review queue.
